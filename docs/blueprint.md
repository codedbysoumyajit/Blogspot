# **App Name**: DevSpace

## Core Features:

- Admin Login: Secure login for administrators to manage blog posts.
- Dashboard: Intuitive dashboard for creating, editing, and deleting blog posts.
- Post Creation: Create new blog posts with a title, description, image, and full content.
- Post Editing: Edit existing blog posts to update content and metadata.
- Post Deletion: Delete blog posts from the dashboard.
- Homepage Display: Display blog posts as interactive cards with an image, title, and short description on the homepage.
- Post View: Dedicated page to view the full content of each blog post.

## Style Guidelines:

- Primary color: Light blue (#A8DADC) for a clean and modern feel.
- Background color: Off-white (#F1FAEE) to create a soft, inviting backdrop.
- Accent color: Darker blue (#457B9D) for interactive elements and highlights, ensuring visibility.
- Body and headline font: 'Inter' (sans-serif) for a modern and readable design.
- Use simple, line-based icons for navigation and actions, complementing the modern aesthetic.
- Use a clean, grid-based layout for the homepage and post pages.
- Subtle animations for transitions and interactions, such as card hover effects and page loading.
